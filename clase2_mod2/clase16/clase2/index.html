git y gitgitgitgithubhub

Git

  • Es un software
  • Es mantenido por linux
  • Es un repositorio local
  • Es un software gratuito y de código abierto
  • No tiene un sistema de gestión de usuarios
  • Mínimas herramientas externas
  • Control de versiones

Github

  • Es un servicio
  • Es mantenido por microsoft
  • Es un repositorio remoto
  • Cuenta con una versión gratuita y una de pago
  • Tiene un sistema de gestión de usuarios
  • Amplio mercado de herramientas externas
  • Almacena versiones de git
